# Protocolo Dieta da Selva — Funil Quiz v2.0

## Deploy na Vercel
1. Suba para o GitHub
2. Importe na Vercel → Deploy

## Dev local
```bash
npm install && npm run dev
```

## Personalizar
Edite `app/quiz.js`:
- **CHECKOUT_URL** → link do GG Checkout
- **QUIZ_QUESTIONS** → perguntas do quiz  
- **TESTIMONIALS** → depoimentos

## Fluxo do Funil (10 telas)
1. Landing (gancho + prova social Fogaça)
2. Nome (personalização)
3. Quiz (8 perguntas com nome)
4. Dados corporais (peso, altura, meta)
5. Educacional GLP-1 (ciência)
6. Educacional Ciência (comparativo)
7. Prova Social (5 depoimentos)
8. Análise animada (loading premium)
9. Diagnóstico personalizado (IMC, meta, tempo)
10. Oferta (preço, garantia, CTA)
